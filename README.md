# 💰 Spendwise - Personal Expense Tracker

A beautiful and intuitive expense tracking web application built with Flask that helps you manage your personal finances with ease.

![Expense Tracker Demo](https://via.placeholder.com/800x400?text=Spendwise+Dashboard)

## ✨ Features

- 📊 **Interactive Dashboard** - View all your expenses at a glance with real-time statistics
- 💵 **Quick Add Expenses** - Add new expenses in seconds with category selection
- 📈 **Visual Analytics** - Beautiful pie charts showing your spending by category
- 📅 **Monthly View** - Group and analyze expenses by month with running totals
- 🎨 **Modern UI** - Glass-morphism design with smooth animations
- 📱 **Fully Responsive** - Works perfectly on desktop, tablet, and mobile devices
- 🗑️ **Easy Management** - Delete expenses with a single click
- 💾 **Persistent Storage** - SQLite database for reliable data storage
